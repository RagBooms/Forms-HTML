Forms

Form is a way of inputting multiple data from the user and can give the whole data back in the database.

        The input tags are being used in forms to input the data from the form and whole and make that data useful in backend.
        The form tag is used to start the form format and whole.
<form>

This tag is used to start the form and all those properties which you want to implement in form.

The "post" attribute is used in the form like <form action="post">
